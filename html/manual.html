PictureGrid Help

Opening a picture

To open a picture, either:

  • •Click on the central area, or 

  • •Select File → Open from the menu. 

To open a recent file, you can also select File → Recent… from the menu.

Pasting a picture

To paste a picture into the window:

  • •Copy the picture to the clipboard. 

  • •Select Edit → Paste from the menu, or press Ctrl+V. 

Changing the Grid

Use the “Options” panel on the left to change the grid.

To show or hide the grid, click the check-box next to “Show Grid?”.

To change the number of columns in the grid, change the value in the “Columns” box.

To change the number of rows in the grid, change the value in the “Rows” box.

To change the color of the grid, click the box next to “Grid Color:”. This will show a color picker window. Choose the desired color and click “Select”.

To change the width of the grid lines, change the value in the “Line Width:” box.

Moving the Grid

To move the position of the grid, click and drag on the picture using the left mouse button.

To move the grid back to its original (center) position, either:

  • •Double-click on the picture, or 

  • •Select Grid → Center Grid from the menu. 

To undo or redo moving the grid, select Edit → Undo or Edit → Redo from the menu.

Saving the Picture

To save the picture with the grid applied, select File → Save As… from the menu.
